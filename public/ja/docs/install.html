インストール

インストールはunit.cssまたはunit.min.cssを読み込むだけです！
CSS本体はGitHubよりクローンもしくはZIP形式でダウンロードするか、npmを経由してダウンロードすることが可能です。

1．GitHubからクローンするか、npm経由でダウンロードする

$ git clone https://github.com/kokushin/units.git
$ npm install unitscss --save-dev

2. unit.min.cssを読み込む

<link rel="stylesheet" href="css/units.min.css">

3. 使用したいコンポーネントのHTMLを記述する

<div class="uc-card _float">
  <div class="uc-card-image" style="background-image:url(https://source.unsplash.com/random)"></div>
  <h3 class="uc-card-title">サンプルカード</h3>
  <p>この文章はダミーテキストです。実際にはこの箇所に任意のテキストが挿入されます。表示を確認してください</p>
  <div class="uc-btn _primary"><a href="#">ダミーボタン</a></div>
</div>

4. 結果を確認してください！
