ヘッダー

ヘッダーコンポーネントは、Webページの最上部に配置される要素を指します。
背景色を変更するオプションや、固定のON/OFFを切り替えるオプションなどが用意されています。

  • クラス一覧
  • 基本のヘッダー
  • テーマヘッダー
  • ゴーストヘッダー
  • 固定ヘッダー

クラス一覧

基本クラスは必須となります。
その他、用途に応じてオプションクラスを指定することができます。

クラス名 種類 説明 必須 指定箇所 併用
uc-header ベース 基本クラス はい - -
uc-header-logo パーツ 基本のヘッダー用ロゴを適用する はい ベースの直下 -
_fixed オプション ヘッダーを固定し、追従するようにする いいえ ベース 不可
_ghost オプション ヘッダーの背景を透過する いいえ ベース 不可
_dark テーマ ヘッダーの背景にダークテーマを適用する いいえ ベース 可能
_black テーマ ヘッダーの背景にブラックテーマを適用する いいえ ベース 可能

基本のヘッダー

ベースとなるヘッダーです。

Logo

Page description

<header class="uc-header">
  <div class="uc-header-logo">Logo</div>
  <p>Page description</p>
</header>

テーマヘッダー

ヘッダーの背景を各テーマごとに変更することが可能です。

Dark

Page description

<header class="uc-header _dark">
  <div class="uc-header-logo">Dark</div>
  <p>Page description</p>
</header>

固定ヘッダー

ヘッダーの位置をページのスクロールに合わせて追従するように固定します。
ちょうど、このドキュメントページのヘッダーのような挙動になります。
