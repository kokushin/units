カード

カードコンポーネントは、情報をわかりやすくひとまとめにしたり、目立たせたりすることができます。
全部で7種類のバリエーションが用意されており、用途に応じて使い分けることが可能です。
横並びにする場合は.uc-gridと組み合わせて使用してください。
