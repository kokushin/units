インストール

インストールはunit.min.cssを読み込むだけです！
CSS本体はGitHubよりクローンもしくはZIP形式でダウンロードするか、npmを経由してダウンロードすることが可能です。

1．GitHubからクローンするか、npm経由でダウンロードする

$ git clone https://github.com/kokushin/units.git
$ npm install unitscss --save-dev

2. unit.min.cssを読み込む

<link rel="stylesheet" href="css/units.min.css">

3. 使用したいコンポーネントのHTMLを記述する

<div class="uc-card-unit">
    <div class="uc-card">
      <h3 class="uc-card-title">Lorem ipsum dolor.</h3>
      <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aspernatur, et.</p>
    </div>
  </div>

4. 結果を確認してください！

Lorem ipsum dolor.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aspernatur, et.
