ヘッダー

ヘッダーコンポーネントは、Webページの最上部に配置される要素が含まれます。
背景色を変更するオプションや、固定のON/OFFを切り替えるオプションなどが用意されています。

  • クラス一覧
  • 基本のヘッダー
  • テーマヘッダー
  • ゴーストヘッダー
  • 固定ヘッダー

クラス一覧

基本クラスは必須となります。
その他、用途に応じてオプションクラスを指定することができます。

クラス名 種類 必須 併用 説明
.uc-header コンポーネント はい - 基本のヘッダーを適用する
.uc-header-logo コンポーネント はい - 基本のヘッダー用ロゴを適用する
._of-fixed オプション いいえ 不可 ヘッダーを固定し、追従するようにする
._of-ghost オプション いいえ 不可 ヘッダーの背景を透過する
._of-dark オプション いいえ 可能 ヘッダーの背景にダークテーマを適用する
._of-black オプション いいえ 可能 ヘッダーの背景にブラックテーマを適用する
