Welcome to UNITS

はじめに

UNITS（ユニッツ）は、シンプルで美しいコンポーネントセットです。
CSSを読み込み、UNITS Componentとして定められたHTMLをコピー&ペーストするだけで誰でも簡単に利用することができます。

素早く利用できます

units.cssを読み込むだけ。あらかじめ様々なコンポーネントが用意されているので、必要なものだけを選択しあなたのWebサイトに組み込むことが可能です。

シンプルで美しいビジュアル

各コンポーネントは美しくデザインされています。デザインは定期的にアップデートされますが、大きな変更は加えず、細かいマージンの調整を中心に行われます。

モダンブラウザサポート

UNITSはMicrosoft Edge, Google Chrome, Firefox, Safariなど各ブラウザの最新バージョンに最適化されています。古いブラウザの動作は...ごめんなさい。

レスポンシブWebデザイン

UNITSはレスポンシブWebデザインを採用し、全てのデバイス上で同じデザインを表示できるように開発を進めています。

定期アップデート

UNITSはまだベータ版なので、短いスパンでアップデートされる予定です。それは新しいコンポーネントの追加やソースコード最適化、デザインの微調整などです。

日本語をサポート

開発者は日本人なので、日本語でのサポートが可能です。日本語で書かれているWebページへの最適化、ドキュメントのアップデートも進めています。
