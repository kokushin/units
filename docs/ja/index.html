Features

UNITS（ユニッツ）は、シンプルで美しいコンポーネントセットです。
CSSを読み込み、UNITS Componentとして定められたHTMLをコピー&ペーストするだけで
誰でも簡単に利用することができます。

素早く利用できます

units.cssを読み込むだけ。あらかじめ様々なコンポーネントが用意されているので、必要なものだけを選択しあなたのWebサイトに組み込むことが可能です。

シンプルで美しいビジュアル

各コンポーネントは美しくデザインされています。デザインは定期的にアップデートされますが、大きな変更は加えず、細かいマージンの調整を中心に行われます。

モダンブラウザサポート

UNITSはMicrosoft Edge, Google Chrome, Firefox, Safariなど各ブラウザの最新バージョンに最適化されています。古いブラウザの動作は...ごめんなさい。

Usage

UNITSは簡単にインストールでき、すぐに使い始めることが可能です！

1. CSS本体はGitHubよりクローンもしくはZIP形式でダウンロードするか、npmを経由してダウンロードすることが可能です。

$ git clone https://github.com/kokushin/units.git
$ npm install unitscss --save-dev

2. unit.min.cssをあなたのWebページで読み込ませてください。

<link rel="stylesheet" href="css/units.min.css">

3. 定められたHTMLを組み、uc-からはじまるコンポーネントクラスを付与して下さい。例えば下記のサンプルでは、カバー画像カードのコンポーネントを適用させることができます。

<div class="uc-card _of-float">
  <div class="uc-card-image" style="background-image:url(https://source.unsplash.com/random)"></div>
  <h3 class="uc-card-title">Sample card</h3>
  <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aspernatur, et.</p>
  <div class="uc-btn _of-primary"><a href="#">Button</a></div>
</div>

4. 表示をブラウザ上で確認してみてください！
